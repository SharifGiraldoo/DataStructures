## 📝 Descripción General

El curso **Estructura de Datos** forma parte del plan de estudios de **Ingeniería de Sistemas y Computación** de la Universidad del Quindío, y se ubica como continuación natural del ciclo de programación (Lógica de Programación → Programación I → Programación II → Programación III → **Estructura de Datos**).

En este repositorio se recopilan los **ejercicios, prácticas y actividades evaluativas**, organizados por núcleos temáticos, con el objetivo de:

- Comprender e implementar las principales **estructuras de datos lineales y no lineales**.
- Desarrollar competencias en el análisis de **complejidad algorítmica**.
- Aplicar estructuras de datos en la resolución de problemas reales del software.
- Promover el uso de **herramientas de control de versiones** como Git/GitHub en el ámbito académico.
- Fortalecer habilidades en **programación orientada a objetos** mediante lenguajes como Java.

Este espacio está diseñado como recurso de consulta, práctica y referencia tanto para estudiantes como para docentes.

## ⚙️ Instalación y Ejecución

A continuación se presentan los pasos para clonar y ejecutar los contenidos del repositorio en un entorno local:

### 1. Clonar el repositorio

```bash
git clone https://github.com/SharifGiraldoo/DataStructures.git
cd DataStructures
```

### 2. Requisitos previos

Asegúrese de tener instalado:

- **Java Development Kit (JDK) 8 o superior**

Verificación:
```bash
java -version
javac -version
```

### 3. Compilar y ejecutar un archivo Java

```bash
# Compilar
javac NombreArchivo.java

# Ejecutar
java NombreArchivo
```

## 📖 Contenido Académico por Núcleo Temático

El curso se organiza en **3 núcleos temáticos** que abarcan desde los fundamentos hasta estructuras avanzadas:

### 🔹 Núcleo 1 — Fundamentos y Estructuras Lineales
> Introducción a la complejidad algorítmica, arreglos, listas enlazadas simples, dobles y circulares, pilas y colas.

### 🔹 Núcleo 2 — Estructuras No Lineales
> Árboles binarios, árboles de búsqueda binaria (BST), árboles balanceados (AVL), montículos (heaps) y tablas hash.

### 🔹 Núcleo 3 — Estructuras Avanzadas y Algoritmos
> Grafos, recorridos (BFS/DFS), algoritmos de ordenamiento avanzados y aplicaciones integradoras.
